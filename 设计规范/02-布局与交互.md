## 状态机(AgentStatus → 状态行左侧)

源码 `setStatus(kind, tool)`。两个已截图确认,其余推断(⚠️ 首帧快照确认):

| AgentStatus | glyph + label(左侧) | 色 |
|---|---|---|
| `Idle` | `◇ 就绪` | `text.muted` |
| `CallingModel` | `(spinner) 调用模型…`(⚠️ 推断) | `accent.primary` |
| `ExecutingTool(name)` | `(spinner) 执行 {name}…`(⚠️ 推断) | `accent.primary` |
| `WaitingForPermission` | `▲ 等待授权…` | `warning.fg` |

状态行**右侧**(常驻):`provider · model │ iter X/maxIter │ N msgs │ cwd`
其中 `N msgs = user|assistant|tool 块计数`(源码 `countMsgs()`),与「轮次 iter」是两个量。

## 交互 / 键位

源码:`window` 上的 `keydown` 监听 + 权限动作行。

| 键 | 语境 | 行为 |
|---|---|---|
| `y` / `Enter` | 权限确认 | 允许 |
| `n` / `Esc` | 权限确认 | 拒绝 |
| 文本 | 输入框 | 编辑输入 |
| `/` 前缀 | 输入框 | slash 命令:`/help` `/clear` `/model [name]` `/status` `/exit` `/login` `/logout` |
| —(自动) | transcript | 新内容自动滚到底(`scrollTop = scrollHeight`) |

> 滚动 / 翻页 / 输入历史(Up / Down / PageUp)等键位原型未覆盖;1.0 实现自定,纳入快照后锁定。
