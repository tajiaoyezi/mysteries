# 01 · 设计令牌(Design Tokens)

> 双主题(Midnight 深色 / Daylight 浅色),**语义结构一致、明暗反转**。
> 落点:`tui/theme.rs` 调色板模块 + 一个断言 token 值的单测(配色漂移 = 测试红)。

## 排版(终端语境)

终端等宽,无字体/字号自由度 → 原型的 font-weight / letter-spacing / font-size **塌缩**为 ratatui `Modifier`:

| 原型表现 | 终端映射 |
|---|---|
| font-weight 700(标题/工具名) | `Modifier::BOLD` |
| 降灰(dim 次要文字) | `Modifier::DIM` |
| letter-spacing(`MYSTERIES` wordmark) | ⚠️ 用空格手动拉开 |
| 选中 / hover 高亮 | ⚠️ `Modifier::REVERSED` 或 accent 底 |

## Glyph(图标字符)

✅ port(终端字体含即可,否则 ⚠️ 降级 ASCII):

| glyph | 含义 | glyph | 含义 |
|---|---|---|---|
| `✦` | 品牌标记 | `◇` | Idle 就绪 |
| `✓` | 工具成功 / 通过 | `▸` `>` | 输入 prompt / 用户消息 |
| `✗` | 工具失败 | `⋯` | 截断省略 |
| `▲` | 警告 / 需授权 / 等待授权 | `↓` `→` `⏎` | 提示 / 回车 |

## 终端能力降级(⚠️ adapt)

- truecolor → 终端仅 256 色时映射最近 `Color::Indexed`;仅 16 色再降一档。1.0 默认 truecolor,降级表归入路线图 1.4(TUI 体验)。
- 圆角 / 阴影 / 渐变 / 动画 → 见 `README` 三分类:统一 ❌ drop 或 ⚠️ adapt 为 box-drawing。
