# Snake-Rogue —— 由 mysteries 生成的验证 demo

> 本目录是一个**可运行的验证产物**:一个贪吃蛇 + 肉鸽(Roguelike)单文件 HTML 小游戏,**全程由 mysteries 自己写出来**——用于证明本 agent 能从需求出发、经 plan 审批、逐步交付一个真正可玩的软件。

## 运行

浏览器直接打开 `index.html` 即可(零依赖、无需构建)。

## 玩法

单文件静态页面,暗色霓虹风格。核心:基础贪吃蛇 + 过关三选一强化卡牌 + 多种食物/道具 + 层数递进 + Boss 层 + 环境修饰符等肉鸽元素。键盘方向键 / WASD 控制。

## 它是怎么来的

由 mysteries 在 **plan 模式**下生成:先只读调研 → `submit_plan` 交出带每步验收的结构化开发计划 → 用户批准 → 逐步执行(`update_plan` 上报进度)。生成过程的截图见 [`../mysteries截图/`](../mysteries截图/)(img_3 ~ img_6:思考、ask_user 选型、计划审批、执行进度面板)。
